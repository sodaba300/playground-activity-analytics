종합 분석

1. 고레벨 적합 콘텐츠 (GT ~ Leaders / MAG · MGT 계열)
상대 참여율 · 고레벨 편중
Activity 기준 분석에서 상대 참여율 기준으로 고레벨에서 눈에 띄게 높은 참여율을 보인 콘텐츠는 아래와 같습니다.
  • 문법, 듣기, 추론, 쓰기, 말하기 유형의 문제가 복합적으로 제시되며
  • UI 구조가 단순하지 않고
  • 사용자가 가이드와 동작 흐름을 스스로 유추하며 진행해야 하는 특징을 가지고 있습니다.
따라서 자기주도적 탐색과 복합 사고가 가능한 고레벨 학습자에게 적합한 콘텐츠로 판단됩니다.
주요 콘텐츠
  1. Magic Word World
  2. Emoji Story Chain
  3. Voice Actor
  4. Treasure Hunter
  5. Magic Fortune English
2. 중간레벨 적합 콘텐츠 (ECP 후반 ~ GT / Honors 계열)
참여 안정성 · 중간레벨
Activity 기준 분석에서 상대 참여율이 대부분의 레벨에서 고르게 나타난 콘텐츠 중, 특히 중간레벨에서 안정적인 사용 패턴을 보인 콘텐츠는 아래와 같습니다.
  • 동작 프로세스가 비교적 단순하고
  • UI 디자인이 직관적이며
  • 문제 난이도가 과도하게 높지 않아 다양한 레벨에서 무난하게 접근 가능한 특징을 보입니다.
특히 중간레벨 학습자의 참여 안정성과 반복 사용 측면에서 적합한 콘텐츠로 판단됩니다.
주요 콘텐츠
  1. Escape the Mystery Room
  2. Sell It!
  3. Story Builder
  4. This is Me!
  5. Birthday Invitation Maker
  6. News Creator
3. 저레벨 적합 콘텐츠 (ECP 계열)
ECP · 상대 참여율
Activity 기준 분석에서 ECP 계열 레벨에서 상대 참여율이 눈에 띄게 높게 나타난 콘텐츠는 아래와 같습니다.
  • 디자인이 단순하고

  • 쉬운 문제 위주 또는 힌트를 통해 정답 유추가 가능한 구조를 가지고 있습니다.
